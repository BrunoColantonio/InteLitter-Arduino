# Arduino

This Arduino have differents Sensors and Actuators, working together to identify when a cat get into their litter box and also when the humans have to clean up the litter box.

## Sensors
- Distance Sensor
- Humidity Sensor
- Button

## Actuators
- Servomotor
- Led RGB
- Display LCD
